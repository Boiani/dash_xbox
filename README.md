Xbox Sales & Performance Dashboard
Este projeto consiste em um Dashboard interativo desenvolvido inteiramente no Microsoft Excel, focado na análise de dados do ecossistema Xbox. O objetivo é transformar dados brutos em insights visuais sobre vendas, performance e métricas de mercado.

Visão Geral do Projeto
O projeto foi estruturado utilizando as melhores práticas de organização de planilhas, separando a interface do usuário da lógica de dados:

Aba Dashboard: Interface visual com gráficos dinâmicos e indicadores de performance (KPIs).

Aba Cálculos: Camada intermediária onde os dados são processados, utilizando fórmulas avançadas e tabelas dinâmicas.

Aba Bases: Onde residem os dados brutos (dataset) utilizados para alimentar a análise.

Aba Assets: Elementos visuais, ícones e referências de design utilizados para manter a identidade visual da marca Xbox# dash_xbox
